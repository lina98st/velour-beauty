# Velour Beauty

![Velour Beauty Preview](assets/preview.png)

A premium beauty studio website for Velour Beauty, a lash and nail studio based in Berlin, Germany.

## Tech Stack

- [React](https://react.dev/) with [Vite](https://vitejs.dev/)
- [Tailwind CSS v4](https://tailwindcss.com/)
- [Framer Motion](https://www.framer.com/motion/)

## Features

- Fully responsive design
- Smooth scroll navigation
- Framer Motion animations and transitions
- Leopard print aesthetic with purple and black color palette
- Booking request form
- Sections: Hero, Services, About, Pricing, Booking, Contact, Footer

## Getting Started

### Prerequisites

- Node.js v20.19+ or v22.12+
- npm

### Installation

Clone the repository and install dependencies:

\`\`\`bash
git clone https://github.com/lina98st/velour-beauty.git
cd velour-beauty
npm install
\`\`\`

### Development

Start the local development server:

\`\`\`bash
npm run dev
\`\`\`

Open [http://localhost:5173](http://localhost:5173) in your browser.

### Build

\`\`\`bash
npm run build
\`\`\`
